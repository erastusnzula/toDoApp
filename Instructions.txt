Installation and Running the App
===============================

1.	Clone the repository:
    git clone https://github.com/erastusnzula/toDoApp.git 
    cd toDoApp/toDoApp

2.	Install dependencies:
    npm install

3.	Start the app using Expo:
    npx expo start
    
4.	Use the QR code in your Expo Go app to preview the app on a physical device or use an emulator.
